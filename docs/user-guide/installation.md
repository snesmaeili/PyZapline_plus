# Installation

## From PyPI (recommended)

```bash
pip install pyzaplineplus
```

## From source

```bash
git clone https://github.com/SinaEsmaeili/PyZaplinePlus
cd PyZaplinePlus
pip install -e .
```

## Optional extras

```bash
# MNE integration
pip install pyzaplineplus[mne]

# Development
pip install pyzaplineplus[dev]
```

## Requirements

- Python 3.8+
- NumPy, SciPy, scikit-learn, Matplotlib

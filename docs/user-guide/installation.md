# Installation

## From PyPI (recommended)

```bash
pip install pyzaplineplus
```

## From source

```bash
git clone https://github.com/snesmaeili/PyZapline_plus.git
cd PyZapline_plus
pip install -e ".[dev]"
```

## Optional extras

```bash
# MNE integration
pip install pyzaplineplus[mne]

# Development
pip install pyzaplineplus[dev]
```

## Requirements

- Python 3.8+
- NumPy, SciPy, scikit-learn, Matplotlib
